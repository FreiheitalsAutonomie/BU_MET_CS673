Overview

The 'communicaiton tool' is a fully-functional communication application. Currently, it supports video and text communication, as well as file sharing through a native system as well as through and integration with Google drive.

Features

  • Text Chat

    Create private or public chat rooms using the 'create new teams' button in the upper right. Messages are perserved between sessions. Each chat room can have an unlimited number of users. Switch between teams by clicking on the team name in the left hand column. Add emoji to your sessions by clicking on the small 'plus' button to the right of the message text bar. For access to special characters hold down a letter key (such as 'o') untill the special character menu appears.

  • Video Chat

    Join a video chat between other members of the chat room. Press the video button while a member of a chat room in order to open the chat for that room. When you leave a room, your video connection will be closed.

  • File Sharing

    Upload files to the server using the file folder icon next to the message input bar. Uploaded files are stored in an unencrypted format on the file server.

  • Google Drive Integration

    Using the Google Drive icon located next to the message input bar, you can upload files to Google Drive and share them with your team!
